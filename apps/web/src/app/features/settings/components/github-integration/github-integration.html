rocket_launch Setup Guide

  1. 1
    Copy the Payload URL Use the copy button in the Webhook Configuration card.
  2. 2
    Open your GitHub repository → Settings → Webhooks → Add webhook Do this for each repository you want to track in Shipyard.
  3. 3
    Set Content type to application/json
  4. 4
    Paste GITHUB_WEBHOOK_SECRET into the Secret field This value is in your server's .env file.
  5. 5
    Select events: push, workflow_run, deployment, deployment_status Or choose "Send me everything" if unsure.
  6. 6
    Make sure your Shipyard service's Repository URL matches the GitHub repo URL exactly Shipyard matches incoming events to services by URL. Example: https://github.com/org/repo
  7. 7
    (Optional) Auto-deploy via GitHub Deployments API In your GitHub Actions workflow, use the deployment and deployment_status events to let Shipyard automatically create and complete deployment records — no custom API calls needed.
